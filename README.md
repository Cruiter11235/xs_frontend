## 💡 Introduction
本项目为2024四川大学计算机学院大学生创新创业院级项目，项目前端采用`Vue3`作为基本前端框架开发，项目后端采用`django`作为基本后端框架开发，
采用ChatGLM3-6B作为AIGC模型来对《全宋文》进行信息抽取，注入neo4j知识图谱中来对古文献进行脉络分析。
